# mylovehtml
本人纯小白，一行代码不会，因为找不到符合需求的网站所以自己用ai写了一个 ，找到的所谓的表白网源码也太丑了，所以写了个非常简单的静态页面，还在逐步完善中...


目前实现的功能 雪花飘落的粒子特效 音乐播放 图片轮播（点击切换）日期计时，呈现效果和放的照片尺寸有关，粒子效果在手机端显示较好，因为本质上是个计时网页。


放上来希望可以激励到自己，也是记录自己瞎折腾的过程。
